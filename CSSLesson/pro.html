↑にあるようなタブ+ページ切り替えをHTMLとCSSのみで作ってみよう！

条件！

  • 色、形はなんでもOK
  • リキッドレイアウトであること
  • CSS+HTMLのみで作成すること

HTMLの雛形


<!DOCTYPE html>
<html lang="ja">

<head>
    <title>Document</title>
</head>

<body>
    <input name="tab" type="radio" checked>
    <input name="tab" type="radio">
    <input name="tab" type="radio">
    <ul>
        <li><label>タブ1</label></li>
        <li><label>タブ2</label></li>
        <li><label>タブ3</label></li>
    </ul>
    <content>
        ページ1
    </content>
    <content>
        ページ2
    </content>
    <content>
        ページ3
    </content>
</body>

</html>

キーワード

  • リキッドレイアウト
  • inputタグ+labelタグの動作
  • 疑似要素
  • 疑似クラス
  • calc
  • 変数
  • デフォルトのCSS書き換え
  • float
